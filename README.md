# Expanding Card

A simple **Expanding Card UI project** built using **HTML, CSS, and JavaScript**.
When a user clicks on a card, it expands smoothly while the other cards shrink, creating an interactive and visually appealing layout.

## 🚀 Live Demo

https://yash880888.github.io/Expanding-Card/

## 🛠 Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, layout, and animations
* **JavaScript** – Card click interaction and expanding functionality

## ✨ Features

* Interactive **clickable cards**
* Smooth **expanding animation**
* Clean and simple **UI design**
* Lightweight project using **pure HTML, CSS, and JavaScript**

## 📁 Project Structure

```
Expanding-Card
│
├── index.html
├── style.css
└── script.js
```

## ⚙️ How It Works

* Multiple cards are displayed in a horizontal layout.
* When a card is clicked:

  * The selected card **expands**.
  * Other cards **shrink**.
* JavaScript handles the **active class toggle** to control the expanding effect.
